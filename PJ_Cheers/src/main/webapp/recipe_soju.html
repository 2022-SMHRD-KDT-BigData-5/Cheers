메로나 소주 칵테일

base : 소주

♡ 준비물 ♡
사이다, 소주, 메로나, 소주잔(계량컵), 세팅잔

♥ 제조방법 ♥
1. 세팅잔에 소주 1~2잔을 넣는다.
2. 사이다는 소주의 2배로 넣는다.
3. 섞은 음료에 메로나를 쏘옥~ 넣는다.
4. 메로나를 콩콩 눌러서 녹이면 끝!
